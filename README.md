## Approach #1: `indexOf()` and `splice()`

indexOf()去找元素的索引，

splice()删除元素，直接改变原数组。

const newArr=splice():返回被删除的数组给newArr[]

~~~jsx
    console.log("splice start");
    const arr = [0, 5, 1, 2, 3];
    const idx = arr.indexOf(5);
    arr.splice(idx, 1);
    const newA = arr.splice(idx, 1);
    console.log(arr);
    console.log(newA);
    console.log("splice end");
~~~
